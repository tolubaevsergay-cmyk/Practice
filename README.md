Основные команды:
/start	Начать общение с ботом
/name: Имя	Представиться боту
/numbers: Числа  Ввести числа, которые будет считать бот
/set: переменная = значение Задать боту переменную и ее значение
/stop  Закончить общение с ботом
число, число	Ввести числа через запятую, пробел или ;
число..число	Задать диапазон чисел (например, 3..9)
ans, число  ans - переменная, которая содержит последний результат вычисления. Можно посчитать ввести в чат-бот и посчитать с числами
переменная, число  Посчитать заданную переменную с числами
+	Сложение введённых чисел
-	Вычитание введённых чисел
*	Умножение введённых чисел
/	Деление введённых чисел
min	Найти минимальное число
max	Найти максимальное число
avg	Найти среднее арифметическое
gcd	Найти НОД (наибольший общий делитель)
